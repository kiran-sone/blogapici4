# Blog Posts REST API in CodeIgniter

## About Blog Posts API application

This is a simple REST API/web application developed using CodeIgniter 4.6 and MySQL. <br>

### Application modules:
- Form to Create/Edit Post
- List Posts
- Pagination
- Buttons to Edit/Delete Post

## Application Requirements

- CodeIgniter version 4.6
- Configure/update .env or .htaccess as per your local environment setup
- MySQL database (blogapi_ci4.sql - included in project root dir)

### Application Access:
http://localhost:8080/
